\begin{abstractutfpr}
This Final Project describes the refactoring, modernization, and expansion of the E-Museu web system, a digital platform for cataloging and disseminating artifacts from the history of computing. The motivation was grounded in technical limitations of the previous version, including maintenance difficulties, lack of automated tests, and the need for more reproducible and secure environments. The work aimed to improve the internal structure of the system, increase stability and sustainability, and implement functional improvements to the catalog. The methodology combined software engineering practices such as MoSCoW prioritization, documented Gitflow, Docker and Docker Compose for services, automated deployment with Coolify on UTFPR domains, refactoring supported by PHPUnit tests, and static analysis and formatting tooling in the repository. Main outcomes include upgrading the PHP/Laravel stack, separating staging and production environments, adding an image gallery, internationalizing content, integrating Cloudflare Turnstile anti-bot protection, and generating QR codes for item identification. In conclusion, the improvements align the E-Museu with contemporary web development practices and strengthen technological memory preservation on a clearer operational basis for future evolution.
\end{abstractutfpr}
